# Change Log

## 0.2.0

- Add syntax highlighting for annotations: `@name` (runtime decorator) and `@!name` (compile-time, processed then stripped).
- Add snippets for `enum`, `data class`, and `switch`/`case`/`default` blocks.
- Add annotation snippets (`@`, `@!`, and the built-in `@!print_on_call`).

## 0.1.0

- Initial release: syntax highlighting, snippets, and LSP integration for `.spc`
  files (interfaces, abstract/final/static modifiers, enums, data classes,
  generics, and scientific notation).
